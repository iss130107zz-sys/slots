# Sweet Bonanza 2500 消除老虎机模拟器

这是一个基于 Python 编写的 Sweet Bonanza 老虎机数学模型模拟器，用于计算理论 RTP（返还率）和消除机制模拟。

### 📌 核心规则摘要
* **消除机制（Tumble）**：滚轮停止后，所有中奖组合将消失，上方符号下落填补空位，直到没有新的中奖组合为止。
* **任意位置中奖（Scatter Pays）**：屏幕上出现相同符号达到一定数量即可中奖，无需连接成线。
* **免费旋转（Free Spins）**：当屏幕上出现 4 个或更多scatter符号时，触发免费游戏，并伴有随机翻倍符号。
